SISTEMA-COMPONENTES

ARDUINO UNO

Procesador

Será nuestra placa madre que utilizaremos para toda la elaboración de nuestro sistema y proyecto elegido y sus diferentes conectores que nos facilitará durante el proceso del armado de nuestro sistema.

SENSOR DE HUMEDAD YL-69

Entrada

Este sensor de humedad del suelo se utilizará para medir la humedad de la tierra para nuestra planta. Los sensores integrados en el sistema de riego ayudan a programar el suministro y la distribución de agua de forma mucho más eficiente.

SENSOR DE TEMPERATURA DHT-11

Entrada

Este sensor de temperatura se utilizará para medir la capacidad de la humedad y un termistor para medir el aire circundante, mostrando los datos mediante una señal digital en el pin de datos.

PULSADOR ROJO

Entrada

El pulsador lo utilizaremos para interrumpir o enviar una señal electrónica de manera momentánea. Para encender o apagar el sistema de una manera más sencilla y rápida de forma manual hacia nuestro sistema.

JUMPER CABLES

Conector

Estos cables se utilizarán para facilitar la conexión hacia nuestro protoboard permitiendo abrir o cerrar nuestro circuito durante la elaboración de este proyecto.

PROTOBOARD

Conector

Nuestra placa de pruebas se utilizará para insertar fácilmente componentes electrónicos en el prototipo de nuestro circuito electrónico y también que nos ayudará en nuestras pruebas para el proyecto y así obtener un mayor resultado.

BOMBA DE AGUA Sumergible DC 12V de 800l/h

Salida

La bomba de agua un dispositivo electromecánico totalmente sumergiblese que se utilizará para transformar e incrementar energía, aplicándola para mover el agua.

PVC MANGUERA Transparente de 1m

Salida

Utilizaremos una manguera, un tubo transparente flexible para facilitar y unir con nuestra bomba de agua para que sea más facil controlarlo. Y para poder aplicarlo a nuestro sistema de riego automático.

RELÉ SSR-25 DA

Salida

Este dispositivo electromagnético lo Utilizaremos para poder controlar cargas AC controladas con señales DC. Y que se encargan de abrir y cerrar el paso de la corriente eléctrica y son accionados bajo este mismo tipo de energía.

VÁLVULA SOLENOIDE 220VAC

Salida

Este dispositivo lo Utilizaremos para controlar el flujo de líquidos en este caso, controlar la cantidad de agua que necesitaremos para el cuidado de la planta. Este componente puede ser opcional o no para nuestro sistema.

CÁMARA ESP32-CAM

Salida

Este dispositivo lo utilizaremos por su Cámara IP de videovigilancia, es un controlador con cámara para transmitir imágenes de tu robot móvil. Su formato DIP permite su fácil y rápida integración en cualquier aplicación y montaje en protoboard.

BATERÍA 9v

Salida

La batería es apropiado para conectar y alimentar directamente a nuestro arduino, lo que hace que sean fáciles de usar.

PANTALLA LCD

Salida

La pantalla LCD lo utilizaremos para la visualización de información, los carácteres y símbolos de una forma gráfica. En este caso veremos y mostrará la información de nuestra humedad, temperatura, condición de la tierra de nuestra planta.
